More Mathjax goodies

Use specific macros inside your latex code to style specific elements in your equation and apply fragments!
  • \texclass[class=any_class]{element} to style an element with any_class
  • \texfragment[index=any_index]{element} to make the element an appearing fragment with data-fragment-index=any_index
  • \fragapply[class=any_class, index=optional_index]{element} to change the style of the element with optional data-fragment-index = optional_index
$\texapply[class=success, index=2]{s}= \frac{\texclass[class=info]{\alpha_0}}{\texapply[class=success, index=2]{4}} + \sqrt{\texapply[class=bg-danger, index=1]{\frac{\sum(x_{i}-\bar{x})^2}{n-1}}} - \int_{a}^{b} \texapply[class=bg-warning, index=1]{x^2} \texfragment[index=0]{dx}$

Fragments fun

Emphasize text by using some special classes in fragments:
  • .highlight-[color]: triggers definitive font color change
  • .highlight-current-[color]: triggers temporary font color change
  • .highlight-bg-[color]: triggers definitive background color change
  • .highlight-current-bg-[color]: triggers temporary background color change
  • .highlight-bg-[color]-white: triggers definitive background color change, with white font color
  • .highlight-current-bg-[color]-white: triggers temporary background color change, with white font color

.fragment .highlight-danger

.fragment. highlight-current-info

.fragment .highlight-bg-success

.fragment. highlight-current-bg-warning

.fragment .highlight-bg-pop-white

.fragment. highlight-current-bg-danger-white
